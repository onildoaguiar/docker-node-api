# How to use this example

##You can run using Docker Compose:

```console
$ docker-compose up -d
```

##Development environment

```console
$ docker-compose -f docker-compose.development.yml up -d
```

##Test environment

```console
$ docker-compose -f docker-compose.test.yml up -d
```
